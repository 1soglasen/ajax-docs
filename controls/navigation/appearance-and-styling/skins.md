# Skins



The following article describes how to set a preferable Skin to the RadNavigation control.

## 

__RadNavigation__ uses __skins__ to control its overall look-and-feel. A skin is a set of images and a CSS stylesheet that can be applied to the control elements (items, images, etc.) and defines their appearance.

To apply a skin to a __RadNavigation__ control, set its __Skin__ property.
