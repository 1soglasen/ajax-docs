# Animation Object



## 

The __Animation__object is returned by the __get_expandAnimation()__ or __get_collapseAnimation()__ method of the [RadToolBar]({%slug toolbar/client-side-programming/radtoolbar-object%}) client object. The following table lists its methods:


| Name | Parameters | Return Type | Description |
| ------ | ------ | ------ | ------ |
| __get_duration__ |none|integer|Returns the duration, in milliseconds, of the animated effect.|
| __set_duration__ |integer|none|Sets the duration, in milliseconds, of the animated effect.|
| __get_type__ |none|enumeration|Returns the type of the animated effect.|
| __set_type__ |enumeration|none|Sets the type of the animated effect.|
