# ImageTile Object



## 

This article explains the client-side properties and methods __[RadImageTile]({%slug tilelist/tiles/image-tile%})__ exposes.

>note  __RadImageTile__ inherits the __[RadBaseTile]({%slug tilelist/tiles/client-side-programming/basetile-object%})__ type and has the API it provides in addition to the ImageTile specific methods.
>



>caption 	RadImageTile Client-Side Methods

| Name | Parameters | Return Type | Description |
| ------ | ------ | ------ | ------ |
|get_imageUrl()||String|Returns the URL of the image dispalyed in the content container.|
|set_imageUrl(value)|String||Sets the URL of the image dispalyed in the content container.|
