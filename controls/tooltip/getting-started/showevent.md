# ShowEvent



## 

The RadToolTip __ShowEvent__ property controls the type of event that triggers the showing of a tooltip. The possible values are:

* __OnMouseOver__(Default): RadToolTip appears on mouse over the target control

* __OnClick__: RadToolTip appears on clicking with the left button of the mouse on the target control

* __OnRightClick__: RadToolTip appears on right click with the mouse on the target control

* __OnFocus__: RadToolTip appears on focus of the target control

For a live demo of this property see [Show On Event](http://demos.telerik.com/aspnet-ajax/ToolTip/Examples/ShowEvent/DefaultCS.aspx).
