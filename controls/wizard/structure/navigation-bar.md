# Navigation Bar



__NavigationBar__ is a wizard element that contains buttons like elements that provides the user with a quick-launch user interface to randomly reach any step of the wizard.

## 
>caption Figure 1: RadWizard Navigation bar

![wizard-navigation-bar](images/wizard-navigation-bar.png)

The NavigationBar position is controlled by the __NavigationBarPosition__ property. This property value accepts one of the following values: __Top__, __Bottom__, __Left__ or __Right__

>caution NavigationBar and ProgressBar can be only placed either on the same side or on opposite sides of the RadWizard control.
>
