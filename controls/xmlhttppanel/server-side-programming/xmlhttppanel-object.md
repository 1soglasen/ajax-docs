# XmlHttpPanel Object



## 

__RadXmlHttpPanel__ provides the following server-side properties:


>caption Â 

| Property | Description |
| ------ | ------ |
| __LoadingPanelID__ |Gets or sets the ID of the RadAjaxLoadingPanel control that will be displayed over the control during the partial page update.|
| __Value__ |Gets or sets a string value depending on which a certain content is loaded in the RadXmlHttpPanel.|
| __WebMethodPath__ |Gets or sets a string value that indicates the virtual path of the WebService used by the RadXmlHttpPanel.|
| __WebMethodName__ |Gets or sets a string value that indicates the WebService method used by the RadXmlHttpPanel.|
| __EnableClientScriptEvaluation__ |Gets or sets a boolean value indicating whether or not the client scripts loaded by the Telerik controls hosted inside the RadXmlHttpPanel should be executed.|
| __RenderMode__ |Gets or sets a value that indicates how the content of an RadXmlHttpPanel control will be wrapped on a page|
