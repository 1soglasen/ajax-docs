# exportToPdf



## 

Telerik RadGrid can export to PDF format.


|  **exportToPdf()**  |
| ------ |
||

Example:

````JavaScript
	        function ExportGrid() {
	            var masterTable = $find("<%=RadGrid1.ClientID %>").get_masterTableView();
	            masterTable.exportToPdf();
	
	        }  
````
