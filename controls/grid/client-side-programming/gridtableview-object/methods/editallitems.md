# editAllItems



## 

Method which switches all GridDataItems in edit mode.


|  **editAllItems()**  |
| ------ |
||

Example:

````JavaScript
function EditAllRecords() {
    var masterTable = $find("<%= RadGrid1.ClientID %>").get_masterTableView();
    masterTable.editAllItems();
}  
````
