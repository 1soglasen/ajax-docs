# insertItem



## 

Method which inserts new table row to the grid. The new data will be taken from the insertion form editors fields.


|  **insertItem()**  |
| ------ |
||

Example:

````JavaScript
function AddNewItem() {
    var masterTable = $find("<%= RadGrid1.ClientID %>").get_masterTableView();
    masterTable.insertItem();
} 
````
