# Skins



__RadCloudUpload__ uses __skins__ to control its overall look-and-feel. A skin is a set of images and a CSS stylesheet that can be applied to the control elements (items, images, etc.) and defines their look and feel. To apply a skin to a __RadCloudUpload__ set its __Skin__ property, either using the properties pane or the control's Smart Tag.

__RadCloudUpload__ is installed with a number of preset skins:
