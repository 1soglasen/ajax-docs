# RibbonBarButtonStrip



## RibbonBarButtonStrip

The __RibbonBarButtonStrip__ has a few important properties, which are listed below:



* __Text__ is the string that the user sees displayed on the __RibbonBarButtonStrip__.

* __ToolTip__ is the text of the tooltip that appears when the user hovers the mouse over the button.

>note Setting the tooltip can enable an accessibility feature: Special accessibility readers like JAWS can pronounce the tooltip of the highlighted item.
>


* __CssClass__ is the name of a CSS style to be applied to the __RibbonBarButtonStrip__. The style can be declared in the CSS file of the applied skin.
