# enum Telerik.Web.UI.ImageEditor.FlipDirection

## Fields

### None `0`

No flipping is done

### Vertical `1`

Image is flipped vertically

### Horizontal `2`

Image is flipped horizontally

### Both `3`

Image is flipped both horizontally and vertically
