# Telerik.Web.UI.GridHTMLExportingEventArgs

The event arguments passed when Telerik.Web.UI.RadGrid exports with HTML format.
            In cases of Word export and Excel with Format set to HTML.

* System.Object
* System.EventArgs
* Telerik.Web.UI.GridHTMLExportingEventArgs
