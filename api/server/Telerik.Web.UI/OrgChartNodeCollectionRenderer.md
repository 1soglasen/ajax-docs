# Telerik.Web.UI.OrgChartNodeCollectionRenderer

Represents the renderer of Telerik.Web.UI.OrgChartNodeCollection OrgChartNodeCollection.

#### Remarks
Renders OrgChartNodeCollection.
            All renderers are attached to the control's tree during PreRender stage.

## Inheritance Hierarchy

* System.Object
* System.Web.UI.Control
* System.Web.UI.WebControls.WebControl
* Telerik.Web.UI.OrgChartNodeCollectionRenderer

## Properties

###  Level `Int32`

Gets or sets depth level of the Node towards RadOrgChart.

###  TagKey `HtmlTextWriterTag`

The default HtmlTextWriterTag is overrided to ul.

#### Remarks
The base HtmlTextWriterTag is span.
