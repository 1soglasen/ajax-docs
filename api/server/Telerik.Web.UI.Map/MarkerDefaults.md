# Telerik.Web.UI.Map.MarkerDefaults

The default options for all markers.

## Inheritance Hierarchy

* System.Object
* Telerik.Web.StateManager
* Telerik.Web.UI.Map.MarkerDefaults

## Properties

###  Shape `String`

The default marker shape. The following pre-defined marker shapes are available:Marker shapes are implemented as CSS classes on the marker element (span.k-marker). For example "pinTarget" is rendered as "k-marker-pin-target".

###  TooltipSettings `Tooltip`

Default Kendo UI Tooltip options for this marker.
