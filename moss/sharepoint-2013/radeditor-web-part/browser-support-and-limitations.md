# Browser Support and Limitations



## 

RadEditor for ASP.NET AJAX for SharePoint supports the following browsers:

* Internet Explorer 6+

* Firefox 2.0+

* Safari 3.0+

* Opera 9.0+

* Google Chrome 2.0+

For more information, you may review the [online browser support section](http://www.telerik.com/products/aspnet-ajax/getting-started/tech-sheets/browser-support.aspx).
